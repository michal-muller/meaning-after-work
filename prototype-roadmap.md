# Meaning After Work — Prototype Roadmap

This document summarizes the current public prototype layer of Meaning After Work.

The prototypes are experimental reflective tools. They are not diagnostic instruments, psychological tests, therapeutic tools or productivity assessments.

## Current prototypes

### The Human Threshold

A short replayable reflective game about staying human under pressure.

Current function:

- timed challenge
- score
- levels
- personal best stored only in the visitor’s browser
- collapse risk reflection

Next improvements:

- expand event library
- tune difficulty
- improve mobile play
- test whether the scoring feels fair and replayable

### Sisyphus Deck

A card-based reflective experience about AI, work, judgment and burden.

Current function:

- five cards
- four reflective dimensions
- final reflective profile
- copy reflection option

Next improvements:

- refine card language
- add more possible card paths
- test whether profiles feel meaningful

### Carrying the Stone

A visual metaphorical experience about work, usefulness, burden and human worth.

Current function:

- five scenes
- animated stone and slope metaphor
- burden, meaning, agency and co-being scores
- final reflection

Next improvements:

- refine visual style of the stone
- make the human figure more expressive
- tune the burden logic

### Contradiction Labyrinth

A decision path through five rooms of contradiction.

Current function:

- five rooms
- four traces: resignation, shortcut, revolt and co-being
- final reflective profile

Next improvements:

- add more rooms
- make the path feel more labyrinth-like
- test whether trace categories are intuitive

### Meaning Blocks

A pattern game about arranging AI, work, care, rest, responsibility and pressure.

Current function:

- grid-based placement
- block families
- neighbourhood logic
- pressure, meaning, agency and co-being scores
- final reflective profile

Next improvements:

- explain scoring more clearly inside the game
- add more block types
- make pattern feedback more visible
- tune replayability

### Contradiction Diary

A slower private writing tool for reflecting on a contradiction.

Current function:

- private browser-based reflection
- no data stored or sent
- copy reflection option

Next improvements:

- refine prompts
- add printable reflection
- consider a separate research contribution flow only with explicit consent

## General design principles

The prototypes should invite reflection rather than diagnose the user.

Preferred language:

- reflective experience
- pattern
- profile
- invitation
- trace
- question
- tendency

Avoid:

- diagnosis
- personality type
- mental health result
- productivity assessment
- validated score
- ranking of human worth

## Data ethics

The current public prototypes are designed to run in the visitor’s browser.

They do not use a database.

They do not collect personal data.

The Human Threshold stores only a personal best score in the visitor’s own browser through localStorage.

No sensitive personal data, medical information, confidential workplace information or third-party information should be entered into any prototype.
